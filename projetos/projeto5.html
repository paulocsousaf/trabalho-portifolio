Calculadora em Python

Disciplina: Lógica de Programação

Sobre o projeto

Calculadora feita no terminal com Python, para praticar variáveis, condicionais e operações matemáticas.

Atividade avaliativa da disciplina. O usuário escolhe a operação (+, -, *, /) e informa dois números para ver o resultado.

O que foi aprendido:

  • Entrada e saída no console
  • Estruturas if/else
  • Tratamento de divisão por zero
