Monitor de Vídeos para vMix

Disciplina: Projeto Integrador

Sobre o projeto

Ferramenta criada para ajudar em transmissões ao vivo que usam o software vMix. A ideia foi monitorar o status dos vídeos e das playlists durante o evento.

Projeto mais prático do curso, ligado a automação e uso de ferramentas reais fora da sala de aula.

Funcionalidades principais:

  • Lista de vídeos da playlist
  • Indicação de qual vídeo está ativo
  • Interface simples para o operador
