CRUD de Alunos

Disciplina: Banco de Dados

Sobre o projeto

Sistema simples de cadastro de alunos feito para praticar CRUD (criar, ler, atualizar e excluir) com banco de dados.

Trabalho entregue em dupla/trio na disciplina. A tela permite adicionar aluno, listar todos, editar dados e remover registro.

Operações do sistema:

  • Cadastrar novo aluno
  • Listar alunos
  • Editar informações
  • Excluir aluno
