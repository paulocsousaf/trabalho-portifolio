## Tema escolhido

**Portfólio de projetos desenvolvidos durante o curso**

O site **ADS em Código** reúne os principais trabalhos feitos nas disciplinas de ADS, em vez de apresentar uma “empresa” ou equipe de tecnologia.

---

## Descrição do projeto

Este projeto é um **portfólio responsivo** feito com **apenas HTML5 e CSS3** (sem JavaScript, Bootstrap ou Tailwind).

O objetivo é mostrar projetos acadêmicos, as tecnologias usadas em cada um e um formulário de contato. Layout **mobile first**, com media queries para celular, tablet e desktop.

## Principais desafios

1. **Foco no conteúdo do curso** — organizar projetos de disciplinas diferentes em um só site.
2. **Menu fixo sem JavaScript** — navegação só com HTML e CSS.
3. **Layout responsivo** — cards que empilham no celular e ficam em grid no desktop.
4. **HTML semântico** — uso de `header`, `main`, `section`, `article` e `footer`.
5. **Trabalho em grupo** — juntar os projetos de cada integrante no mesmo portfólio.

## Como visualizar localmente

1. Baixe ou clone o repositório.
2. Abra `index.html` no navegador.
3. Ou use Live Server no VS Code.

---

## Publicar no GitHub Pages (no final do trabalho)

1. Crie o repositório no GitHub.
2. Envie os arquivos da pasta `portfolio`.
3. Em **Settings → Pages**, escolha branch `main` e pasta raiz.
4. Copie o link gerado para este README.
